Direct Download

To access, follow these instructions,

1. Click on release v1
2. Download "CloudController.app.zip" NOT "Source code (zip)"
3. Open the app once unzipped
4. Click "Done" on the pop-up
5. Open system settings and navigate to "Privacy and Security"
6. Scroll down to the bottom of this section.
7. Look for the message stating "CloudController was blocked..."
8. Then authenticate the app with a password or Touch ID.
9. Finally, reopen the app after completing the prompt.


//It is required to authenticate the app because I do not have a paid Apple Developer account.

Using the app

Click on any control you want to remap and follow the on-screen instructions.
Make sure to also follow the instructions in the setup tab
If a bridge error is present, just try the setup; the error can change after the connection is found.
If the error persists but the controller works fine, this error can be safely ignored.
If wanting to use this tool on other sites, just copy and paste the code from the setup page in the app into the console section on any other site.
